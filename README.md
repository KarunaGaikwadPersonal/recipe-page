## Overview

### The challenge

Users should be able to:

- See a responsive recipe card that works on mobile and desktop.
- View recipe details including preparation time, ingredients, instructions, and nutrition.
- Navigate the page easily with accessible landmarks and focus states.
- Enjoy hover and focus effects for interactive elements.

## My process

### Built With

- Semantic HTML5
- CSS3 (Flexbox)
- Google Fonts (Inter)
- Responsive design principles
- [Frontend Mentor](https://www.frontendmentor.io/) challenge platform

### What I learned

- How to use **main and section landmarks** for better accessibility.
- Implementing **overlay text on images** while keeping responsive scaling.
- Creating **responsive cards** with flexible width, padding, and font sizes.
- Styling **tables, lists, and ordered steps** for clear presentation.
- Using **hover and focus-visible states** for links.

### Continued Development

- Enhance **dark mode** or theme toggling.
- Add **animations** to instructions (e.g., step highlighting).
- Integrate **JavaScript filtering or recipe variants** for interactive recipes.
